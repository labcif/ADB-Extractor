# ADB-Extractor
Python Script to extract Private, Public and APK File from Android emulator or device using ADB

## Requirements
- Python 3
- ADB
- Android Emulator or Device
- tar 
- gzip

## Usage

```bash
python3 acquisition.py -a com.example.app -d emulator -t private
```
